НАУЧИТЬСЯ УЧИТЬСЯ
=============================
Этот одностраничный сайт - 
учебный проект по HTML и CSS, выполненный в ходе обучения в Яндекс.Практикуме

РЕШЕННЫЕ ЗАДАЧИ
-----------
- Сайт сверстан в соотвествии с макетом
- Соблюдена семантика тегов
- Для создания сеток использован flex
- Реализована анимация из брифа
- Ссылкам задан псевдокласс :hover
- Реализована вставка видео через iframe и YouTube API

ЗАПУСК
-----------
Для локального запуска:
- перенести ветку мастер на локальную машину
- запустить из корня ветки мастер файл index.html
